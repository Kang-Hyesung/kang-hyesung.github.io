### 1. New Teams 클라이언트 배포 & 트러블슈팅

최신 시험 트렌드인 **New Teams(2.0)** 관련 문제가 매우 자주 출제된다.

| 증상 / 시나리오 | 원인 및 해결 방법 (정답 키워드) |
| --- | --- |
| **VDI 배포 시 대역폭 최소화** | **MSIX Package** 다운로드 후 로컬 경로 지정 (`-p`) |
| **설치 오류: "Organizational policy"** | 레지스트리 `HKLM\...\Appx`에서 **`AllowAllTrustedApps`** 제거/수정 |
| **설치 오류: "We've run into an issue"** | **Windows 10**일 경우 **WebView2 Runtime** 수동 설치 필요 |
| **Outlook 모임 추가 기능 안 보임** | 클라이언트 재설치 X → **Meeting Policy** (AllowOutlookAddIn) 확인 O |
| **전화 수신 불가 (발신은 됨)** | **Call answering rules** (통화 응답 규칙) 확인 (Forwarding 해제) |

### 2. 네트워크 & 대역폭 (Magic Numbers)

계산기를 두드리기보다 **기준 숫자**를 외워서 직관적으로 판단해야 한다.

* **포트 (Ports)**
* 오디오/비디오/화면 공유 트래픽: **UDP 3478-3481** (TCP 443 아님!)


* **대역폭 (Bandwidth) - 최소 요구량**
* **Screen Sharing (Meetings)**: **250 kbps** (가장 헷갈리는 수치, 암기 필수)
* **Video (HD/Recommended)**: **1.5 Mbps** ~ 2.5 Mbps
* **Teams Rooms (Best)**: **4 Mbps**


* **시험장 판별법 (Rule of Thumb)**
* **유선(Wired)**: 대역폭 넉넉함 → 대부분 **Yes**
* **무선(Wireless) + Video**: 사용자 수 × 1.5Mbps 계산 시 보통 초과됨 → **No**



### 3. VDI vs Physical (인프라 추천)

"VDI를 우선 사용하되, 특정 기능이 필요하면 어떻게 할래?" 유형이다.

* **시스템 오디오 공유 (Share System Audio)**
* VDI에서 **지원 불가** (기술적 제약).
* 정답: **Traditional Desktop (Physical)**


* **사용자 지정 배경 (Custom Backgrounds)**
* 최신 **Azure Virtual Desktop (AVD)**에서는 지원됨.
* 정답: **Azure Virtual Desktop**



### 4. 디바이스 관리 (Teams Rooms, Panels, Displays)

하드웨어 장비 설정은 메뉴 위치가 정해져 있다.

* **화면/동작 설정**: 배경화면, 예약 연장(Extend), 조기 종료(Release), 절전 모드 등
* 정답: **Configuration Profile (구성 프로필)**


* **원격 재부팅/업데이트 권한**
* Teams Communications Admin (X) → 통화/번호 관리용
* **Teams Device Administrator (O)** → 장치 관리용



### 5. 컴플라이언스 & 정책 (Security)

키워드 매칭으로 1초 만에 풀어야 하는 문제들이다.

| 요구 사항 | 정답 솔루션 |
| --- | --- |
| **부서 간 대화 차단** (Block communication) | **Information Barriers** (IB) |
| **욕설/성희롱/민감어 감지** (Detect text) | **Communication Compliance** |
| **정책 우선순위 (Rank)** | **숫자가 작을수록 우선** (Rank 1 > Rank 3) |
| **타운홀(Town Hall) Copilot** | **주최자(Organizer) & 발표자(Presenter)**만 사용 가능 |

### 💡 합격을 위한 마지막 팁

1. **"Policy" 구분하기**: 기능이 안 되면 **Meeting Policy**, 앱이 안 보이면 **App Setup Policy**, 전화가 안 되면 **Voice/Calling Policy**.
2. **UDP는 국룰**: 음성/화상 품질 문제는 99% **UDP 포트** 차단 문제이거나 네트워크 지연(Jitter/Latency) 문제다.
3. **New Teams = MSIX**: `.exe`나 `.msi`는 구버전 이야기다. New Teams는 무조건 **MSIX**다.
