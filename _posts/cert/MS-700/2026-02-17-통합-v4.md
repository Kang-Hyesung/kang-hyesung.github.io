# 📌 1. [공식] 키워드 매칭 (보면 바로 답 찍기)

문제 지문에서 **왼쪽 키워드**가 보이면 **오른쪽 답**을 찾아라.

| 문제 키워드 (Keywords)                     | 정답 키워드 (Answer)                      | 비고                                       |
| ------------------------------------- | ------------------------------------ | ---------------------------------------- |
| **Headset**, Driver, Device issue     | **Quality of Experience (QoE)**      | 장치 문제는 무조건 QoE                           |
| **Connection Failure**, Dropped Calls | **Failure Drill Down**               | 연결 실패 원인 분석                              |
| **Best Practices**, People experience | **Adoption Score**                   | Usage Report 아님! (Usage는 양, Adoption은 질) |
| **Toll-free number** (수신자 부담)         | **Communication Credits**            | 공짜 전화는 돈(Credit)이 있어야 함                  |
| **Custom Policy Package**             | **Teams Premium License**            | 커스텀 패키지는 유료 기능                           |
| **Sensitivity Label** for Meetings    | **Teams Premium License**            | 회의 보안 라벨도 유료 기능                          |
| **Virtual Receptionist**, Menu        | **Auto Attendant (AA)**              | "1번 누르세요"는 AA                            |
| **FIFO**, Bypass Menu, No Menu        | **Call Queue (CQ)**                  | 줄 서기는 CQ                                 |
| **Emergency Responders** (911)        | **Emergency Address** + **Location** | 주소 만들고 번호에 매핑                            |
| **Existing Invites Update** (초대장 갱신)  | **Start-CsExMeetingMigration**       | MMS라고도 부름                                |

# ⚔️ 2. [비교] 헷갈리는 개념 3대장

시험에서 가장 많이 틀리는 3가지 구분을 완벽하게 정리함.

### ① Policy(정책) vs Settings(설정)

* **Meeting Settings**: **조직 전체(Tenant-wide)** 적용.
* *예: 외부 익명 사용자(Anonymous) 참가 허용, 이메일 초대장 로고.*


* **Meeting Policy**: **사용자(User) 별** 차등 적용.
* *예: User1은 녹화(Recording) 가능, User2는 불가능.*
* *팁: "User1이 ~를 못한다"면 90% Policy 문제.*



### ② Calling Plan vs Direct Routing

* **Calling Plan (MS 요금제)**: 복잡한 라우팅 필요 없음.
* *필요한 것: 라이선스 + 전화번호 + Enterprise Voice 활성화.*
* *함정: **Voice Routing Policy**는 오답임.*


* **Direct Routing (통신사 직접 연결)**: 복잡함.
* *필요한 것: SBC, PSTN Gateway, **Voice Routing Policy**.*



### ③ 권한 레벨 (Least Privilege)

* **Teams Communications Support Specialist**: **Tier 1**. 기본 도구(Basic Tools). 특정 사용자만 조회 가능.
* **Teams Communications Support Engineer**: **Tier 2**. 고급 도구(Advanced Tools). 전체 통화 로그 조회 가능.
* **Teams Administrator**: 전체 관리자 (Too much power).

# 🛠 3. [문제 해결] 트러블슈팅 시나리오

### Q1. 다이얼 패드(Dial Pad)가 안 보여요!

1. **Phone System License** (또는 E5) 할당.
2. **Enterprise Voice Enabled**를 `$true`로 설정.

### Q2. 수신자 부담 번호(Toll-free)가 갑자기 안 돼요!

* **원인**: **Communication Credits** 잔액 부족 (0원).
* **해결**: 크레딧 충전(Add Credits).

### Q3. 전화를 거절(Decline)했는데 음성 사설함으로 안 가요!

* **설정 위치**: **Calls ring me** 설정 확인.
* **세부 설정**: **If unanswered** 항목이 'Voicemail'로 되어 있는지 확인.
* *(참고: 'Busy on busy'는 통화 중일 때만 작동함)*

### Q4. QoS 설정을 했는데 적용이 안 돼요!

* Teams 관리 센터에서 켜는 건 '허용'일 뿐임.
* **진짜 설정**은 **Group Policy (GPO)**에서 포트별 **DSCP 값**을 지정해야 함.
* *Audio: DSCP 46 / Video: DSCP 34*



### Q5. 사용자의 '오디오 회의 번호'를 바꾸고 싶어요!

* **위치**: Teams 관리 센터 > Users > User1 > **Account** 탭.
* *(Voice 탭 아님! Voice는 개인 전화, Account는 회의용 번호)*

# 📊 4. [숫자] 암기해야 할 대역폭

**150 kbps** 환경에서 가능한 것:

1. **Audio** (당연히 됨)
2. **1:1 Video** (딱 150kbps 필요)

* *(안 되는 것: 회의 중 비디오, 화면 공유 - 얘네는 200~250kbps 이상 필요)*

# 💡 5. 튜터의 마지막 조언 (문제 풀이 접근법)

1. **주어를 봐라**: "User1만" 문제면 **Policy**나 **User settings**, "회사 전체"면 **Settings**나 **Org-wide**.
2. **돈 냄새를 맡아라**: "Custom", "Advanced Protection", "Toll-free" 단어가 나오면 **Premium 라이선스**나 **Credit**이 정답.
3. **순서를 기억해라**:
* 전화 개통: 라이선스 → 번호 할당 → 엔터프라이즈 보이스 켬.
* 긴급 전화: 주소 생성 → 번호에 주소 입힘.


4. **PowerShell vs UI**:
* 초대장 업데이트는 무조건 **PowerShell** (`Start-CsExMeetingMigration`).
* 사용자 설정 변경은 주로 **UI** (Admin Center).
